SETBASERATEIf there is a filter immediately following this command, it will be used to set the baseline firing rate; using time window in xex rather than baseline window in menu. If this stored rate exists, this is what image map will use, so if you want to use usual pre-rf baseline, then CLEARBASERATE first
CLEARBASERATEClear stored baseline firing rate
BINWIDTHFollowing numeric value sets the value of binwidth. Between 0 and 100
